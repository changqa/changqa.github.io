#欢迎来到老安的博客

半年前就在万网上买了 anzail.online ，现在终于有时间把博客搭起来了。
用 hexo 建站，选择了北邮师兄 melody 主题，因为阿里的学生服务器也过期了，索性直接把博客部署到了 GitHub 上，不得不说十分良心。

##记录一下踩过的坑

###图床

本来用微博当图床，奈何图片质量实在糟糕，压缩得不能看，v2ex 上发现了良心免费图床 sm.ms ，可以传 5m 以下的图片，已经免费了三年，作者家里有矿。

###主题的问题

如果直接

git clone -b master https://github.com/Molunerfinn/hexo-theme-melody themes/melody

应该会遇到 pv 访问日志记录功能无法使用的问题，这是因为 melody 的日志统计功能使用了 busuanzi 服务，后者因为域名到期，更换了新的域名，而 melody 主题目前只在 dev 分支更新了，master 分支还是指向了旧的域名，获取不到服务。

解决方法是找到 themes\melody\layout\includes\count\busuanzi.pug 打开后将
dn-lbstatics.qbox.me 改为 busuanzi.ibruce.info 。
