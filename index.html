迷蒙的爆款文章

我是觉得咪蒙这一类的文字工作者挺坏的，最近在朋友圈刷屏又反转的《一个出身寒门的状元之死》，乍看之下，真的很容易收割焦虑。

含咪率的手动检测

新朋友关注咪蒙了吗？

微信在公众号的界面中可以查看此公众号的好友关注数量，因此在添加新好友时可以根据咪蒙公众号的关注数量有没有 ++1 来判断。

老朋友关注咪蒙了吗？

如果朋友没有设置 不能通过手机号添加 ，并且你们之间仅仅是点赞之交，聊天记录并不重要，而且你可以承担误删他无法再找回的风险。
那么可以先把他的好友删除，根据咪蒙公众号的关注数量有没有 --1 来判断。

由于单方面删除好友，再次添加不需要验证，可以在操作之后通过手机号添加回好友。

含咪率检测器

上述操作的原理十分粗暴，所以考虑能否实现自动的含咪率检测。

遇到的问题包括：

  • 聊天记录的备份与恢复
  • 找到再次添加好友的可行方式：手机号、微信号、qq号、群聊etc，排除不能再次添加的好友
  • 。。。。。。

关于功能的脑洞：

  • 支持对指定好友咪态的判定：“他咪了”“她没咪”
  • 生成用户个人含咪率海报“您的好友中 XXX 等56人咪了，充值9.9元，马上查看您的咪态好友”
  • 对于删除后无法再次添加的好友，判定为“薛定谔的咪”，不进行删除
  • 。。。。。。

就差一个程序员了

之前知道有 itChat ，以为至少做到删除好友还挺简单的。结果到 github 上找了一圈，都没有提供删除好友的接口，因为 web 版的微信不支持。

一个感觉能蹭到热度的产品破产了。。。。。。

老安上线了 2019-01-23|扯闲篇儿

欢迎来到老安的博客

半年前就在万网上买了 anzail.online ，现在终于有时间把博客搭起来了。
用 hexo 建站，选择了北邮师兄 melody 主题，因为阿里的学生服务器也过期了，索性直接把博客部署到了 GitHub 上，不得不说十分良心。

记录一下踩过的坑

图床

本来用微博当图床，奈何图片质量实在糟糕，压缩得不能看，v2ex 上发现了良心免费图床 sm.ms ，可以传 5m 以下的图片，已经免费了三年，作者家里有矿。

主题的问题

如果直接

git clone -b master https://github.com/Molunerfinn/hexo-theme-melody themes/melody

应该会遇到 pv 访问日志记录功能无法使用的问题，这是因为 melody 的日志统计功能使用了 busuanzi 服务，后者因为域名到期，更换了新的域名，而 melody 主题目前只在 dev 分支更新了，master 分支还是指向了旧的域名，获取不到服务。

解决方法是找到 themes\melody\layout\includes\count\busuanzi.pug 打开后将
dn-lbstatics.qbox.me 改为 busuanzi.ibruce.info 。
