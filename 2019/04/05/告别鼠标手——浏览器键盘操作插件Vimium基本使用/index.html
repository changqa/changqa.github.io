告别鼠标手——浏览器插件Vimium基本使用

为祖国服务60年

coder 全身上下，除了脑子，最重要的可能就是手了。

前一段时间手机和电脑操作的时间比较长，手腕不自觉会疼，特别是右手。分析认为，键盘操作引起的劳损还在其次，最主要是鼠标操作太伤手腕了。于是寻找一种告别鼠标的解决方案，争取解放右手，摆脱鼠标。

需求分析

windows 下开发，ide 主要用 sublime 和 jetbrain 全家桶，浏览器用火狐。

需要掌握以上环境或软件的常用快捷键操作，当然需要一步一步来，养生 这个 tag 下面会连载几篇相关的文章，也方便我自己查询。

正题：浏览器操作神器 Vimium

Vimium 是以 vim 的风格操作浏览器的插件，firefox 和 chrome 浏览器都有，原理其实是快捷键绑定，所以修改起来也很方便。

以工作环境中最常见的操作，新标签页打开百度 --> 在输入框中搜索问题 --> 打开相关页面浏览回答--> 关闭页面 为例，介绍常见的 vimium 操作。留意 Vimium 操作中区分大小写。

  • O （大写）打开一个新标签页
  • o （小写）在当前的标签页上打开
  • gi 将光标移到当前第一个输入框
  • f （小写）显示当前页面上各个链接的操作标签，输入后从当前页面直接跳转
  • F （大写）显示当前页面上各个链接的操作标签，输入后打开新页面跳转
  • j 浏览页面下翻
  • k 浏览页面上翻
  • h 浏览页面左滑
  • l 浏览页面右滑
  • shift + j 切换到左一个标签页
  • shift + k 切换到右一个标签页
  • shift + h 当前标签页后退
  • shift + l 当前标签页前进
  • x 关闭当前标签页
  • X 恢复刚刚关闭的标签页
  • ? 不记得指令没有关系，任何标签页下输入 ? 都可以呼出查询指令功能
  • ……

实际使用中遇到的问题

  1. 在 csdn 网页使用可能有一定卡顿，特别是使用 f 指令，如果没有及时弹出黄色标签，稍等就好
  2. 网上教程中 zo zi 分别是界面的放大缩小，实际使用中好像不行……
