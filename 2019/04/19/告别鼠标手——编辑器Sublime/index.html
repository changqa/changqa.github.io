2019-04-19

学不会Vim

隔壁工位小哥 hhkb 的机械键盘着实好看，空白键帽也很秀，可惜我感觉 vim 对新手开发者太不友好了，还是继续用 Sublime 。

便签页操作

和 firefox 等浏览器上的标签页操作基本一致：

  • Ctrl + w : 关闭当前标签页
  • Ctrl + Tab : 下一标签页
  • Ctrl + Shift + T : 打开上次关闭的标签页
  • Ctrl + o : 从新标签页打开
  • Ctrl + n : 打开新标签

编辑操作

  • Ctrl + l : 选中当前行，再次按会选中下一行
  • Ctrl + Shift + D : 另起一行，并复制当前所选中的部分
  • Ctrl + Shift + L : 多行光标，在当前选中的每一行行尾出现编辑光标
